## 阶段一: 云服务基础设施

### Task 1.1: MQTT Broker (Mosquitto)

**目标:** 在云服务器上运行 Mosquitto，配好认证和 TLS

**步骤:**
1. 安装 mosquitto + mosquitto-clients
2. 创建密码文件，添加 desk 用户
3. 配置 TLS 证书(自签)，监听 8883 端口
4. 启动并验证 pub/sub 双向通信

**验证:** `mosquitto_pub -h localhost -p 8883 -u desk -P <pass> --cafile ca.crt -t desk/test -m "hello"`

**产出:** `/etc/mosquitto/conf.d/desk.conf`, systemd enabled

---

### Task 1.2: ASR 服务 (百炼 Paraformer 流式)

**目标:** 提供 HTTP 端点接收音频 chunk，返回流式识别文字

**文件:**
- 创建: `services/asr_service.py`
- 测试: `tests/test_asr_service.py`

**接口:** `POST /asr/stream` 接收 multipart audio chunks → SSE 流式返回 `{"text": "今天", "is_final": false}`

**验证:** `curl -F "audio=@test.wav" http://localhost:5002/asr/stream` → 返回识别文字

---

### Task 1.3: TTS 服务 (Edge-TTS)

**目标:** 文字转语音，返回 MP3 文件 URL

**文件:**
- 创建: `services/tts_service.py`
- 测试: `tests/test_tts_service.py`

**接口:** `POST /tts/synthesize` body JSON `{"text": "..."}` → 返回 `{"url": "/tts/audio/xxx.mp3"}`

**验证:** curl 请求 → 能下载播放 MP3

---

### Task 1.4: 天气服务 (和风天气)

**目标:** 定时拉取天气数据，通过 MQTT 推送到设备

**文件:**
- 创建: `services/weather_service.py`
- 测试: `tests/test_weather_service.py`

**逻辑:** 每 30 分钟拉取和风天气 API → 检查预警 → 推送 MQTT `desk/display` (常规) / `desk/alert` (预警)

**验证:** 模拟 API 返回 → 验证 MQTT 消息内容

---

### Task 1.5: OTA 固件托管

**目标:** Nginx 静态路由提供固件版本检查和下载

**文件:**
- 创建: `/var/www/firmware/version.json`
- 配置: Nginx location `/firmware/`

**接口:**
- `GET /firmware/version.json` → `{"version": "1.0.0", "url": "/firmware/v1.0.0.bin", "size": 1048576}`
- `GET /firmware/v1.0.0.bin` → 固件二进制

**验证:** curl 访问 → 返回正确 JSON 和 bin 文件

---

### Task 1.6: Hermes 新增意图

**目标:** 在 Hermes Agent 中注册墨水屏相关能力

**意图列表:**
- `web_search` — 语音搜索(复用 Firecrawl)
- `set_timer` — "30分钟后提醒我" → cron job
- `set_reminder` — "明天9点开会" → 数据库 + cron job
- `customize_daily` — "日报加嵌入式板块" → 修改日报 cron 配置
- `weather_query` — "今天天气" → 和风天气 API
- `schedule_query` — "今天日程" → 数据库查询

**文件:**
- 创建: `services/hermes_intents.py` (或 plugin)
- 测试: `tests/test_hermes_intents.py`

---

## 阶段二: ESP32 固件骨架 + 驱动

### Task 2.1: ESP-IDF 项目脚手架

**目标:** 创建可编译的 ESP-IDF 项目，点亮串口打印

**文件:**
- 创建: `firmware/CMakeLists.txt`, `firmware/main/CMakeLists.txt`, `firmware/main/main.c`
- 配置: `firmware/sdkconfig` (PICO-V3, 240MHz, 4MB Flash, PSRAM 8MB)
- 分区表: `firmware/partitions.csv` (按设计方案)

**验证:** `idf.py build` → 成功; `idf.py flash monitor` → 串口输出 "Ink-Player boot"

---

### Task 2.2: WiFi + MQTT 客户端

**目标:** 上电自动连 WiFi，连上 MQTT Broker，收发消息

**文件:**
- 创建: `firmware/main/wifi.c`, `firmware/main/mqtt_client.c`

**逻辑:** WiFi 预写入固件 → 自动连接 → 连接 MQTT (TLS) → 订阅 `desk/#` → 发布 `desk/online`

**验证:** 串口看到 "WiFi connected" + "MQTT subscribed"

---

### Task 2.3: 墨水屏驱动 (SPI + 局部刷新)

**目标:** 用 SPI 驱动微雪 4.2 寸屏，实现局部刷新

**文件:**
- 创建: `firmware/main/display.c`, `firmware/main/display.h`

**功能:** 初始化 → 清屏 → 局部刷新指定区域 → 4 级灰度支持

**验证:** 全屏显示测试图案 → 局部刷新时间数字

---

### Task 2.4: UI 渲染引擎

**目标:** 根据 JSON 指令渲染屏幕布局

**文件:**
- 创建: `firmware/main/ui_render.c`, `firmware/main/ui_layout.h`

**MQTT 指令格式:**
```json
{
  "type": "full_refresh",
  "blocks": [
    {"type": "time", "x": 0, "y": 0, "w": 200, "h": 40, "value": "14:32"},
    {"type": "weather", "value": {"temp": 28, "icon": "sunny"}},
    {"type": "schedule", "items": [{"time": "10:00", "title": "组会"}]},
    {"type": "alert", "text": "1小时后下雨", "inverted": true}
  ]
}
```

**验证:** 从 MQTT 发 JSON → 屏幕正确渲染各区块

---

## 阶段三: 语音交互链路

### Task 3.1: I2S 音频采集 + HTTP 上传

**目标:** 从 INMP441 采 I2S 音频 → chunked HTTP POST 到 ASR 服务

**文件:**
- 创建: `firmware/main/audio_capture.c`

**验证:** 录音 5 秒 → 上传 → 服务器收到完整音频文件

---

### Task 3.2: ULP VAD 声能检测

**目标:** ULP 协处理器持续监听 → 有声能超过阈值 → 唤醒主 CPU

**文件:**
- 创建: `firmware/main/ulp_vad.c`, `firmware/ulp/main.S`

**验证:** 拍手 → 串口输出 "ULP triggered" → 主 CPU 唤醒

---

### Task 3.3: ESP-SR 唤醒词

**目标:** 加载"小智"唤醒词模型 → 确认后开始录音

**文件:**
- 创建: `firmware/main/wake_word.c`
- 模型: 放入 `firmware/srmodel/`

**逻辑:** ULP 唤醒 → ESP-SR 跑模型 → 置信度 >85% → 启动录音+LED反馈

**验证:** 说"小智" → 串口输出 "Wake word detected" → 开始录音

---

### Task 3.4: 语音对话完整链路

**目标:** 端到端语音交互: 唤醒词 → 录音 → ASR → Hermes → TTS → 播放

**文件:**
- 修改: `firmware/main/main.c` (状态机)
- 创建: `services/voice_pipeline.py` (云端编排)

**验证:** 说"小智，今天天气怎么样" → 屏幕显示 + 喇叭播报
