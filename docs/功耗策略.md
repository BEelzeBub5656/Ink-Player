## 功耗策略: ULP VAD

INMP441 → I2S → ULP 协处理器 (持续, ~2mA)
  → 声能 > 阈值? → 是 → 唤醒 ESP32 主CPU
  → ESP-SR 确认唤醒词"小智"
  → 匹配? → 录音上传 → AI交互 → 休眠
  → 不匹配? → 直接休眠

| 模式 | 功耗 | 说明 |
|------|------|------|
| ULP 监听 | ~2mA | ULP 只做声能检测 |
| 主CPU待机 | ~0.5mA | 屏幕保持, WiFi休眠 |
| 语音交互 | ~150mA | 每次约5秒 |
| **续航** | **~40天** | 2000mAh, 每天10次交互 |
