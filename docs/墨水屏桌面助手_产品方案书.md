## 1. 产品概述

### 1.1 一句话定位

**常亮的AI语音桌面伴侣——信息永远在眼前，不用唤醒屏幕。**

### 1.2 解决什么痛点

| 现有方案 | 痛点 |
|----------|------|
| 小爱触屏音箱 | LCD息屏全黑，看不清时间；全程小米云，隐私存疑；回答受限 |
| 手机看时间/天气 | 解锁→打开App→3步才看到，打断工作流 |
| 传统电子钟 | 只有时间，没有AI，没有语音 |

### 1.3 产品形态

```
┌─────────────────────────┐
│     4.2英寸墨水屏        │  ← 常亮显示，纸质观感
│  ⏰ 14:32  ☀️ 28°C     │
│  📅 今日日程...          │
│  ⚠ 1小时后下雨          │
├─────────────────────────┤
│   木质 / 白色磨砂外壳    │  ← 融入办公桌不突兀
│   电池续航 10-14天       │
│   Type-C 充电 (手机通用)  │
└─────────────────────────┘
```

## 2. 市场定位

### 2.1 目标用户

- 办公桌需要"瞥一眼"获取信息的白领

## 3. 核心功能

### 3.1 信息常显

| 功能 | 刷新频率 | 说明 |
|------|----------|------|
| ⏰ 时间显示 | 每分钟局部刷新 | EPS32本地RTC, 断网也准 |
| 🌤️ 天气 | 每小时 | 云端拉取，墨水屏局部刷新不闪屏 |
| 📅 日程 | 每小时 + 语音触发 | 同步日历，今日事项一览 |
| 📰 日报摘要 | 每日8:00 | AI精选科技+新闻摘要 |
| ⚠️ 天气预警 | 每30分钟 | 主动推送，反色强调 |
| ⏳ 倒计时/番茄钟 | 语音设置 | 本地计时，到点提醒 |

### 3.2 语音交互（离线唤醒+云端AI）

```
用户说话 → ESP-SR本地唤醒词"小智" → 录音 → 云端Whisper转文字 → Hermes AI理解意图 → 返回结果
```

| 能力 | 示例 |
|------|------|
| 闲聊对话 | "小智，讲个笑话" |
| 知识搜索 | "小智，什么是差分信号" |
| 日程查询 | "小智，明天有什么安排" |
| 设置提醒 | "小智，30分钟后提醒我开会" |
| 控制IoT | "小智，关灯" |
| 音乐播放 | "小智，放点轻音乐" |

### 3.3 交互设计原则

```
👁️  被动信息 → 瞥一眼（时间/天气/日程 永远显示）
🗣️  主动交互 → 语音（唤醒词"小智"）

不做滚轮翻页 → 屏幕设计就是"零交互信息展示"
不做触摸屏 → 保持墨水屏低功耗特性
```

## 4. 技术架构

### 4.1 总体架构

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  墨水屏终端    │ ←→  │  云服务器      │ ←→  │  外部服务     │
│  ESP32-S3    │ MQTT │  Hermes AI   │ HTTP │  天气API     │
│  墨水屏+语音  │      │  Whisper ASR │      │  日历API     │
│  电池供电     │      │  MQTT Broker │      │  新闻源      │
└──────────────┘     └──────────────┘     └──────────────┘
```

### 4.2 硬件规格

| 模块 | 型号 | 规格 |
|------|------|------|
| 主控 | ESP32-S3 | 双核 240MHz, 8MB PSRAM, WiFi/BLE |
| 屏幕 | 微雪 4.2寸 | 400×300, SPI, 4级灰度, 局部刷新 |
| 音频输入 | MEMS 模拟麦 | 灵敏度 -42dB |
| 音频输出 | MAX98357A + 3W 4Ω | I2S Class-D, SNR 95dB |
| 电池 | 3.7V 1200mAh | 续航 10-14 天 |
| 充电 | TP4056 | Type-C 5V/1A, 4小时充满 |
| 待机功耗 | ~200μA | （画面保持时） |
| 工作功耗 | ~150mA | （语音交互时，每次约5秒） |

### 4.3 软件架构

```
ESP32 固件层:
├── ESP-SR 唤醒词引擎 (离线)
├── I2S 音频采集 + HTTP chunked 上传
├── MQTT 客户端 (双向通信)
├── 墨水屏驱动 (LUT 局部刷新)
├── A/B 双区 OTA 固件升级
└── 电源管理 (深度睡眠 + 定时唤醒)

云端服务层:
├── Whisper.cpp 流式 ASR (中文)
├── Hermes Agent AI 引擎
├── MQTT Broker (Mosquitto)
├── Edge-TTS 语音合成
└── 定时任务 (天气/日报/预警)
```

| 壁垒 | 说明 |
